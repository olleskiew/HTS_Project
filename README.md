# HTS_Project
This repository contains files related to final project for HTS subject at Jagillonian Univeristy.
The project was done in 01/2026

## About project:
The main goal for our final project is to verify wheter reanalysis of publicly available, old RNA-seq data with today's tools makes sens and can bring new information.

## Used data:
For our project we decided to use reads from BioProject PRJNA147913, due to the fact this reads were uploaded into SRA archive in 2011. We also used human reference genome: GRCh38.p14 form NCBI
